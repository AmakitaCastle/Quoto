方案 3 预览 - 边缘检测纹理背景

从封面提取主题色 + 边缘检测生成抽象纹理

处理流程

1
上传封面
用户选择图片
→
2
提取主色
5 个区域采样
→
3
边缘检测
Sobel 算子
→
4
生成纹理
抽象线条/点
→
5
叠加渲染
渐变 + 纹理 + 遮罩
示例 1: 星空主题 提取 5 色 + 星星纹理
示例 2: 古典主题 提取 5 色 + 纸质纹理
示例 3: 几何主题 提取 5 色 + 几何线条
示例 4: 自然主题 提取 5 色 + 有机纹理
方案 3 优势：
1. 保留封面特征：通过边缘检测提取原图的轮廓和纹理特征
2. 抽象化处理：不是简单模糊，而是生成艺术化的线条/点阵纹理
3. 文字清晰：原图不直接显示，只用颜色 + 抽象纹理 + 遮罩
4. 个性化：每本书的封面生成独特的背景纹理

技术实现：
- Sobel 算子或 Canny 边缘检测
- 可选：霍夫变换提取直线
- 将边缘映射为 Canvas 路径绘制
- 使用提取的主色作为纹理颜色
